> **Version 5.5.3 - Great Vault Reward Alerts**
>
> This update adds visual alerts to the **Vault** module when a weekly reward is available to claim.
> It also fixes alert refresh after claiming rewards and updates TOC versions for Retail and Mists of Pandaria Classic.

### _Retail :_

- 🆕 _**NEW** -_ Added pending reward alerts to the **Vault** module with configurable alert color, flash, snooze, optional chat confirmation, right-click snooze, and tooltip highlighting.
- 🛠️ _**IMPROVEMENT** -_ TOC update for patch 12.0.7.
- 🐞 _**BUGFIX** -_ Fixed **Vault** alert state not clearing immediately after claiming a Weekly Reward.
- 🐞 _**BUGFIX** -_ Fixed **Vault** pending reward tooltip to use Blizzard unclaimed reward strings.

### _Mists of Pandaria Classic :_

- 🛠️ _**IMPROVEMENT** -_ TOC update for patch 5.5.4.
